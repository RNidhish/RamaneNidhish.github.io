Création d'une Base de Données

Étapes :

1. Analyse du Programme National (PN).
2. Conception du Diagramme de Classe.
3. Modélisation de la Base de Données.
4. Implémentation de la BDD.

Résultats Attendus :

Un schéma de base de données bien conçu et une documentation expliquant la structure de la BDD et ses relations avec le PN du BUT informatique.

Apports :

Ce projet nous a permis d'appliquer nos connaissances en modélisation de données et en conception de bases de données dans un contexte réel, renforçant ainsi notre compréhension des concepts de base de données et de leur application pratique.

Statistique

Objectif du projet :

Création d'une base de données à partir d'un schéma UML en langage SQL. Peupler cette base à l'aide d'un fichier CSV. Après avoir créé une base de données et peuplé celle-ci, nous avons étudié ces données. La troisième partie de ce projet consiste à récupérer le fichier CSV et, avec des variables explicatives et une variable endogène, étudier leur corrélation. Ce projet nous permet de valider les apprentissages critiques suivants :

Apprentissages critiques :

1. Mettre à jour et interroger une base de données relationnelle (en requêtes directes ou à travers une application)
2. Visualiser des données
3. Concevoir une base de données relationnelle à partir d’un cahier des charges
